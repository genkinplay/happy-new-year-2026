# 新年快乐 - Happy New Year 2026

一个美观的新年祝福网页，带有烟花、雪花动画和响应式设计。

## 特色功能

- 🎉 烟花动画效果
- ❄️ 飘落的雪花
- 🏮 传统灯笼元素
- ✨ 发光文字效果
- 📱 响应式设计
- 🎨 渐变背景
- 🔥 互动点击效果

## 技术栈

- HTML5
- CSS3 (包含动画和渐变效果)
- JavaScript (动态元素和交互)
